# Punch Up

Punch Up is a social productivity platform focused on consistency, discipline, and self-improvement.

## Features

### Version 1

- Google Authentication
- User Profiles
- Task Management
- Daily Streak System
- Consistency Grid
- Community Posts
- Comments
- Notifications
- Analytics
- Badges

### Version 2

- Follow Users
- Direct Messaging
- Personalized Feed
- Online Presence
- Leaderboards

### Version 3

- AI Planner
- Premium Features
- Profile Viewers
- Advanced Analytics
- Queue System
- Redis Caching

## Tech Stack

Frontend:
- Angular
- TypeScript
- Angular Material

Backend:
- Node.js
- Express.js

Database:
- MongoDB Atlas

Authentication:
- Google OAuth
- JWT

Realtime:
- Socket.io

Storage:
- Cloudinary
